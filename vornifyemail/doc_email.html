VornifyEmail API Documentation

Send Email

POST https://api.vornify.se/api/email

Request Format:

{
    "to_email": "recipient@example.com",
    "subject": "Your Email Subject",
    "html_body": "..."
}

Example Request with Template:

{
    "to_email": "user@example.com",
    "subject": "Welcome to Our Service",
    "html_body": `
        
        
        
            

                

                    
Welcome!

                

                

                    
Hello User,

                    
Thank you for joining our service.

                

            

        
        
    `
}

Success Response:

{
    "status": true,
    "message": "Email sent successfully",
    "messageId": "unique-message-id",
    "timestamp": "2024-02-23T19:04:40.789Z"
}

Important Notes:

  • HTML body must be properly formatted and escaped
  • Use inline CSS for email compatibility
  • Maximum email size: 10MB
  • Rate limit: 100 emails per minute
